Flash a LilyGO T-Dongle S3 into a USB NCM virtual network adapter that serves the pairing page to the computer at 192.168.4.1.

The laptop should see the flashed dongle as a USB Ethernet-style adapter using the host CDC-NCM driver. The dongle assigns the host an address and serves the local pairing UI.
